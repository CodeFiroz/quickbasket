# QuickBasket

QuickBasket is a modern full-stack eCommerce platform built with Next.js, TypeScript, MongoDB, Razorpay, and Express.js.

The project focuses on delivering a seamless online shopping experience with features such as product management, authentication, cart functionality, secure payments, order management, and image handling through a dedicated service.

## Tech Stack

* Next.js
* TypeScript
* MongoDB
* Express.js
* Tailwind CSS
* Razorpay
* JWT Authentication

🚧 Currently under development.
